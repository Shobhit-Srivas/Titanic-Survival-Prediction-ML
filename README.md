# 📖 Project Description

The **Titanic Survival Prediction** project is a Machine Learning classification application that predicts whether a passenger would survive the Titanic disaster based on passenger information such as age, gender, ticket class, fare, family members, and embarkation point.

The project includes an attractive **Streamlit Web Application** where users can:

- 🎯 Predict survival instantly
- 📊 Explore Titanic dataset
- 📈 View model performance
- 📉 Perform Exploratory Data Analysis (EDA)
- 🔍 Analyze passenger information interactively

This project demonstrates the complete Machine Learning workflow from data preprocessing to deployment.

